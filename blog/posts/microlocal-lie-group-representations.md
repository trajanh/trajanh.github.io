## Why Markdown export from Notion?

The Markdown export is usually the most seamless fit for this website because:

1. It is lightweight and easy to edit in Git.
2. It preserves equations that can be rendered with MathJax.
3. It works directly with the blog renderer on this site.

## Example equation

For a smooth test function $f$ and distribution character $\Theta_\pi$, a typical expression is

$$
\Theta_\pi(f) = \int_G f(g)\,\mathrm{tr}\,\pi(g)\,dg.
$$
